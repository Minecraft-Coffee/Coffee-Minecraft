## About

**Coffee Client Toolkit** gives you instant access to battle-tested PvP & utility assets. Activate **ClickGUI**, **ESP**, **KillAura**, **AutoTotem**, load **JS modules**, and use **server configs** — all in one click.

## What's included

- `modules/` — ClickGUI, ESP, KillAura, AutoTotem (JS)
- `configs/` — Server profiles for popular networks
- `presets/` — PvP, Minigame, Survival packs
- `guides/` — 30-second setup, optimization tips
- `assets/` — GUI skins, gameplay screenshots

## Download & install (archive delivery)

1. Click **Launch Coffee Now** — download starts (`.zip`).
2. Extract to `C:\Coffee-Client-Toolkit`.
3. Run `CoffeeLauncher.exe`
4. Launch Minecraft → tools auto-activate.
> Pro tip: Run as admin for full performance.
